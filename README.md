# OpenJraw

OpenJraw는 Java 기반의 AI Agent Runtime 프로젝트입니다.

단순한 챗봇 Wrapper를 만드는 것이 아니라,  
AI Agent, Tool, Skill, LLM을 위한 확장 가능한 Runtime 플랫폼 구축을 목표로 합니다.

현재 프로젝트는 MVP 초기 단계에서 개발 중입니다.

# Why OpenJraw?

최근 AI 생태계는 특정 클라우드 환경과 벤더 중심으로 빠르게 발전하고 있습니다.

OpenJraw는 Local 및 Self-Hosted 환경에서도 유연하게 동작할 수 있는
개방형 AI Agent Runtime 구축을 목표로 합니다.

- Local 및 Self-Hosted 환경 고려 아키텍처
- Skill 기반 Agent 실행 구조
- 확장 가능한 Agent Runtime 설계
- Java 기반 개발 생태계
- OpenAI-Compatible / Local Model Server 연동
- Tool 및 Plugin 확장 시스템

장기적으로는 개발자가 직접 제어하고 확장할 수 있는
오픈 AI Runtime 플랫폼 구축을 목표로 합니다.
